#### ビルドセットアップ
## 必要物
・NodeJS(最新の安定板)<br>
・Yarn(npmコマンドの進化したやつ)

## インストール手順
``` bash
# 必要ライブラリとかインストール
yarn install

# Developerモードで実行
yarn dev

# アプリケーションを.exe実行ファイルに書き出します。（※日本語ユーザー名だと書き出せないよ）
yarn build

# lint あうべての JS/Vue `src/`のなかのファイル構文をチェックします。結構細かいこと言う。。。
yarn lint

```
